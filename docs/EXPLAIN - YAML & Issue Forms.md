# YAML & GitHub Issue Forms (What these .yml files do)

## What is YAML?
YAML (files ending in `.yml` or `.yaml`) is a simple format for structured data:
- **Indentation matters** (spaces, not tabs)
- It’s used to configure tools (like GitHub templates and Actions)

Example:

```yml
name: Weekly Sprint Packet
labels: ["sprint-packet"]
