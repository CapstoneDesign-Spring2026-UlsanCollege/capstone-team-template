# Week 9 Student-Facing File Pack

## Purpose

This folder contains the student-facing materials for **Week 9: Sprint 3 Kickoff — MVP Verification + Code Ownership**.

The main goal this week is simple:

> Prove the MVP is real: run it, explain it, test it, debug it, and link evidence.

1. `01_WEEK_9_CLASS_PLAN.md` — post first in the LMS/class repo.

10. `10_COURSE_ANCHORS_AND_REMAINING_SCHEDULE.md` — remaining schedule/source-of-truth reference.

## Friday submission reminder

Each team submits **one Week 9 Weekly Sprint Packet Issue** by Friday 23:59.

It must include:

- Weekly Progress Demo evidence
- Link to `docs/AI_CODE_OWNERSHIP_AUDIT.md`
- Board snapshot
- Shipped/verified work links
- Bug list with severity
- Risk/blocker list
- AI Use + Code Ownership Check
- Top 3 goals for next week
- Individual Contribution Receipts from every student

## Core rule

AI is allowed.

Mystery code is not.

AI-assisted work only counts if the team can run it, explain it, test it, debug it, and link evidence.
